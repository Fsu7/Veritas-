## 功能要求

### FR-001 [P0] 6个Agent状态标签
每个Agent显示：名称（中文）+ 状态图标（⏳/⚙️旋转/✓/✗）+ 状态文字 + 耗时。

### FR-002 [P0] 状态色三重传达
标签颜色+图标+文字同步变化，确保不仅靠颜色传达信息。

### FR-003 [P1] 响应式布局
lg横排6标签，sm 2×3网格。

### FR-004 [P1] 点击展示中间结果
点击标签展开/popover展示intermediateResult。

### FR-005 [P2] 整体进度条
顶部el-progress显示完成进度。

### FR-006 [P1] 空状态处理
无Agent状态时显示'等待开始分析'。
