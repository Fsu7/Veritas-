## 版本控制说明

### 文档命名规范

```
[序号]-[文档名称].md

示例：
01-项目策划案.md
01-项目策划案-v1.0.md（归档版本）
```

### 版本号规则

| 版本变化 | 说明 | 示例 |
|---------|------|------|
| v1.0 → v1.1 | 小修改（错别字、格式调整） | v1.0 → v1.1 |
| v1.0 → v2.0 | 大修改（内容重构、新增章节） | v1.0 → v2.0 |

### 更新流程

1. 修改文档前，先复制一份并升级版本号
2. 在文档顶部更新"最后更新"日期
3. 在文档末尾添加修改记录
4. 更新本 README 中的"最后更新"列
