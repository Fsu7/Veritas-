# XH-202630 科研文献助手 - 零基础学习路线图

> **项目**：科研文献智能助手（领域知识个性化生成与多智能体协同决策系统）
> **创建日期**：2026年5月23日
> **文档版本**：v1.0
> **说明**：本文档基于项目全部技术文档，为零基础开发者梳理从入门到能参与开发所需学习的全部知识，按学习阶段和优先级排列。

---

## 学习路线总览

```
第0阶段：编程语言基础（2-4周）
├── Python 基础
├── Java 基础
├── JavaScript / TypeScript 基础
└── SQL 基础

第1阶段：开发框架入门（3-5周）
├── Python Web：FastAPI
├── Java Web：Spring Boot 3.x
├── 前端：Vue3 + Element Plus
└── 数据库操作：JPA / Redis / ChromaDB

第2阶段：AI 核心知识（4-6周）
├── 大语言模型（LLM）基础概念
├── Prompt Engineering
├── RAG（检索增强生成）
├── Embedding 与向量数据库
├── LangChain 框架
└── LangGraph 多Agent编排

第3阶段：模型部署与调优（2-3周）
├── Transformers 库与本地模型加载
├── Qwen2 模型使用
├── bge-large-zh Embedding 模型
├── 模型量化（INT4/INT8）
└── 流式输出

第4阶段：工程化与部署（2-3周）
├── Docker 容器化
├── Docker Compose 多服务编排
├── Java-Python 跨语言服务通信
├── 缓存策略与性能优化
└── Git 版本控制

第5阶段：项目实战串联（4-6周）
├── 按功能实现顺序逐模块开发
├── 多Agent协同工作流
├── 个性化引擎
├── 前后端联调
└── 测试与优化
```
